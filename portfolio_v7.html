CubeSat System Design

Category: Systems Engineering
Status: Ongoing
Year: 2025-Present

Project Overview

Comprehensive design of a 3U CubeSat satellite system including power generation, thermal management, and structural components. Integrating multiple engineering disciplines to create a functional small satellite system.

Objective: Design a thermally stable, power-efficient CubeSat capable of operating in low Earth orbit for 2+ years.

System Components

  • Power Subsystem: Solar panel arrays with battery backup
  • Thermal Management: Multi-layer insulation and radiator design
  • Structure: Aluminum frame with vibration isolation mounts
  • Communications: RF communication system for data transmission
  • Propulsion: Miniature cold-gas thruster for orbital adjustments

Technical Analysis

Detailed thermal analysis to predict temperature variations during orbital cycles, power budget calculations to ensure adequate energy availability, and structural analysis to verify frame resilience to launch accelerations.

Physics Simulation Engine

Category: Software Development
Status: Completed
Year: 2024-2025

Project Overview

Custom-built numerical simulation engine for dynamics and orbital mechanics. Python-based tool implementing advanced numerical methods to accurately simulate complex physical systems.

Feature: Supports n-body gravitational systems, orbital mechanics, and rigid body dynamics with adaptive time-stepping.

Key Features

  • Runge-Kutta integration methods for accurate trajectory prediction
  • N-body gravitational force calculation with optimized algorithms
  • Orbital mechanics solvers for Kepler problems
  • Real-time visualization of trajectories and system states
  • Adaptive error control for maintaining numerical accuracy

Applications

  • Predicting satellite orbital decay and re-entry predictions
  • Analyzing three-body problems in gravitational systems
  • Educational tool for understanding orbital mechanics
  • Baseline for comparing commercial simulation software

Control Systems Design

Category: Control Engineering
Status: In Progress
Year: 2025-2026

Project Overview

Development and implementation of feedback control systems for aerospace applications. Designing robust controllers that maintain system stability under various operating conditions.

Goal: Create stabilizing control systems with response times under 100ms and steady-state error less than 2%.

Control Approaches

  • PID Control: Tuning for setpoint tracking and disturbance rejection
  • State-Space Design: Full-state feedback for improved performance
  • LQR Optimization: Optimal control with weighted cost functions
  • Stability Analysis: Root locus and Bode diagrams for system characterization

Tools Used

  • MATLAB and Simulink for simulation and design
  • Control System Toolbox for analysis functions
  • Real-Time Workshop for embedded implementation

Parametric Design Study

Category: Aerodynamic Optimization
Status: Active
Year: 2025-Present

Project Overview

Systematic optimization of aircraft wing geometry using parametric modeling techniques. Exploring the relationship between geometric parameters and aerodynamic performance to identify optimal configurations.

Approach: Automated parameter sweeps and design space exploration to identify Pareto-optimal solutions balancing lift, drag, and structural weight.

Parametric Variables

  • Wing planform area and aspect ratio
  • Airfoil section and camber distribution
  • Sweep angle and dihedral
  • Wing thickness distribution

Optimization Methodology

Multi-objective optimization framework combining aerodynamic analysis with structural constraints. Each design variant is evaluated for aerodynamic efficiency, structural mass, and manufacturing feasibility.

Outcomes

Comprehensive database of wing designs with associated performance characteristics, enabling future aircraft design decisions with quantitative support and understanding of trade-offs.
